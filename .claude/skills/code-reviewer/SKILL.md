$ARGUMENTS 에 대해 코드 리뷰시 항상 포함되어야 할 것들.

1. **비유를 통해설명**: 각 문단별로 수행하는 기능을 일상생활의 비유하여 이해하기
   쉽게 설명.
2. **다이어그램 그려서 제공**: 각 로직의 기능을 다이어그램으로 제공하여 이해를
   돕기.
3. **라인별 코드 설명**: 각 줄마다 어떤 기능을 하는지, 기능상 무의미한 코드라면
   삭제, 필요한 코드라면 설명 및 주석달기.
